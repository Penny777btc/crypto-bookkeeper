# 加密资产记账工具使用指南

## 🚀 快速开始

### 1. 安装依赖
打开终端（Terminal），进入项目目录，运行以下命令安装所需的软件包：
```bash
npm install
```

### 2. 启动项目（需要两个终端窗口）

本项目包含前端界面和本地后端代理（用于抓取真实数据）。你需要同时启动它们。

**第一步：启动后端（在终端窗口 1）**
```bash
cd server
npm install  # 首次运行需要安装依赖
npm start
```
看到 `Local proxy server running at http://localhost:3001` 即表示启动成功。

**第二步：启动前端（新建一个终端窗口 2）**
```bash
# 确保回到项目根目录
cd .. 
# 或者直接打开新窗口进入项目目录
npm run dev
```
启动后，你会看到类似 `http://localhost:5173` 的地址，在浏览器中打开它即可。

## 📖 功能说明

### 1. 币价监控 (Price Monitor)
- **功能**：查看你关注的币种实时价格。
- **操作**：
    - 在下拉框中选择你想监控的币种（如 BTC, ETH）。
    - 点击 "Add Coin" 添加。
    - 价格每 60 秒自动刷新，也可以点击 "Refresh" 手动刷新。

### 2. 中心化交易所资产 (CEX Assets)
- **功能**：记录你在币安、OKX 等交易所的资产。
- **操作**：
    - **API 配置**：输入交易所的 API Key 和 Secret。
    - **获取数据**：点击 "Refresh Data (Real)"，系统会通过本地后端代理安全地从交易所获取你的真实余额。
    - **注意**：API Key 仅在本地传递，不会上传到任何云端服务器。

### 3. 链上钱包 (On-chain Wallets)
- **功能**：追踪你的链上钱包余额。
- **操作**：
    - **添加钱包**：选择链（Ethereum, Solana 等），输入钱包名称和地址。
    - **查看余额**：点击 "Scan Wallets (Real)"，系统会查询链上真实的原生代币余额（如 ETH, SOL）。

### 4. 法币与 USDT 出入金 (Fiat Ledger)
- **功能**：手动记录你的银行卡或 OTC 出入金记录。
- **操作**：
    - 选择类型（入金 Deposit / 出金 Withdraw）。
    - 输入金额、币种（CNY/USD）、平台（如招行、支付宝）。
    - 右侧卡片会自动计算你的总入金、总出金和净投入。

### 5. 交易明细 (Transactions)
- **功能**：核心记账功能，记录每一笔买卖。
- **操作**：
    - **记一笔**：输入时间、类型（买入/卖出）、交易对（如 BTC/USDT）、数量、价格、手续费。
    - **筛选**：可以通过币种或平台筛选历史记录。
    - **统计**：顶部卡片会显示你的总买入量、总卖出量和总手续费支出。

## ⚙️ 高级配置

如果你想添加新的币种或交易所，可以修改 `src/config/config.ts` 文件：

- **添加币种**：在 `supportedCoins` 列表中添加。
- **添加交易所**：在 `platforms` 列表中添加。

## 🔒 数据安全
本项目是**本地运行**的。所有数据（API Key、交易记录）都存储在你浏览器的 `localStorage` 中。
- **优点**：隐私安全，不经过第三方服务器。
- **注意**：如果你清空浏览器缓存，数据会丢失。建议定期备份（目前需手动记录，后续可开发导出功能）。
